# Information Integration Skill

## Overview

This skill provides comprehensive information integration through information collection and information recommendation.

## Quick Start

### Collect news by keyword

```python
# scripts/get_information.py
python scripts/get_information.py --keywords ["keyword1", "keyword2", ...]
```

### Collect news by date

```python
# scripts/get_information.py
python scripts/get_information.py --date "20260112"
```

### Recommend similar content

```python
# scripts/get_recommendation.py
python scripts/get_recomendation.py --content "What is the newest AI news?"
```
